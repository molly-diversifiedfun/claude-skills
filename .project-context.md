# Project Context: claude-skills (Public Skill Library)

## What
Free public repo of Claude Code skills. Research-backed, each with SKILL.md + reference files. Taster menu for paid solopreneur-skills.

## Stack
- Markdown (SKILL.md files)
- No build system — pure skill definitions
- Skills installed to ~/.claude/skills/

## Brand
- Open source / community — no specific brand voice
- READMEs should be clear, practical, developer-friendly

## Status
- 25 working skills across 8 categories
- All 16 PRD'd thinking/writing/career/learning/editing/meeting skills shipped (Batches 1-5 complete as of 2026-05-15)
- Plus 9 marketing/shipping/audit samples from the paid collection
- Individual READMEs for all 25 skills
- Main README with 8 categories

## Key Decisions
- Research-backed: each skill has a PRD in `docs/prds/` built from primary research in `docs/research/`
- Batched development (research → research brief → PRD → build) — parallelized via subagents on the final 8 in 2026-05-15 session
