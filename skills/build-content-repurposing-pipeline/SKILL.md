# Build Content Repurposing Pipeline

## What This Skill Does

This skill is the "create once, distribute everywhere" engine. Give it one piece of long-form content — a blog post, podcast transcript, webinar recording notes, video script, or newsletter — and it produces 8-12 platform-native content pieces, each tailored to the conventions, character limits, and audience expectations of its target platform.

The Content Multiplication Framework turns one piece into up to 10 derivatives:

1. **Twitter/X Thread** (8-12 tweets extracting key insights)
2. **LinkedIn Post** (authority-building thought piece, 150-300 words)
3. **Instagram Carousel** (5-10 slides, each with one key point)
4. **Instagram/TikTok Reel Script** (30-60 second video script)
5. **Email Newsletter** (story-driven email linking back to original)
6. **YouTube Short Script** (60 second script for YouTube Shorts)
7. **Blog SEO Summary** (meta description + key takeaway for search)
8. **Pull Quotes** (5-8 quotable, screenshot-worthy one-liners)
9. **Pinterest Pin Copy** (title + description for visual pin)
10. **Community Post** (discussion-starter for Slack, Discord, or Facebook group)

Every piece is copy-paste ready, platform-native, and can stand on its own — someone who only sees the LinkedIn post gets full value without needing the original.

## Phase 1

Gather the information needed to extract, transform, and distribute the content. Ask these questions ONE AT A TIME. Wait for the user's answer before moving to the next question.

### Question 1

"Share the content you want to repurpose. Paste the full text, a link, or a transcript. This can be a blog post, newsletter, podcast transcript, webinar recording notes, video script, or long-form social post."

If stuck: "Even a rough draft or outline works. What's the core piece you want to multiply across platforms?"

### Question 2

"What platforms do you want to target? Pick from: Twitter/X, LinkedIn, Instagram (carousel + reel), TikTok, Email, YouTube Shorts, Pinterest, Community posts."

If stuck: "Which 3-4 platforms does your audience use most? Start there — you can always add more later."

### Question 3

"What's the ONE key takeaway from this content? If someone could only remember one thing from this piece, what should it be?"

If stuck: "Finish this sentence: 'After reading/watching/listening to this, people should know that...'"

### Question 4

"What's the CTA for each piece? Where should people go after consuming each piece — link to the blog? DM you? Sign up for something? Buy something?"

If stuck: "What's the one action you want people to take? Should it be the same CTA across all pieces, or different per platform?"

### Question 5

"What's your brand voice? Casual, professional, bold, technical, warm — or something else entirely?"

If stuck: "How do you normally talk on social media? Show me a post you liked that you wrote, and I'll match that tone."

If `brand-voice-router` output is available, skip this question and use that voice profile for all pieces.

## Phase 2

If `viral-hook-generator` output exists, use those hooks for the platform pieces. If `generate-persona-playbook` output exists, match platform tone to persona.

### Section 4: Repurposing Gaps and Optimization

Analyze and report on:

- **Content density**: Is the original piece rich enough to support all requested formats? Flag if the source material only has enough substance for 5-6 derivatives instead of 10.
- **Platform gaps**: Which platforms are missing from their strategy that could reach new audiences?
- **Hook variations**: Suggest 2-3 alternative hooks for the top-performing platforms (Twitter hook tweet, LinkedIn opening line, Reel first 3 seconds).
- **Evergreen potential**: Can this content be reshared in 30, 60, or 90 days with fresh hooks? If yes, suggest the re-share angles.
- **Series potential**: Does this piece suggest a natural series? If so, propose the follow-up topics (Part 1 of X).

## Rules

- Each platform piece must be NATIVE to that platform — never copy-paste the same text across platforms.
- Twitter threads need a HOOK tweet that stands completely alone. Most people only see tweet 1.
- LinkedIn posts must NOT start with "I" — lead with the insight, not yourself.
- Instagram carousel slides must each contain ONE idea — not paragraphs.
- Reel and TikTok scripts must hook in the FIRST 3 SECONDS — no slow intros.
- Email newsletters must be story-driven, not link dumps.
- Never fabricate stats, data, or proof points that were not in the original content.
- Every piece must be ready to copy-paste — no [FILL IN] placeholders except for images or visuals that require design.
- Character counts must be accurate — do not write a 400-character "tweet."
- Hashtag limits: Instagram 5-15, LinkedIn 3-5, TikTok 3-5, Twitter 1-2. Never hashtag spam.
- Each piece must stand ALONE — someone seeing only one piece should get full value without needing the original.
- The publishing schedule should create momentum, not fatigue — space pieces out across the week.
- If `brand-voice-router` is active, apply the voice profile to ALL pieces.
- If `humanize-ai-writing` is available, apply it to all generated text.
- If `viral-hook-generator` output exists, use those hooks for platform pieces instead of generating new ones.
- If `generate-persona-playbook` output exists, match platform tone to persona characteristics.

## What This Skill Does NOT Do

- Create visual designs, graphics, or images (use a design tool for carousel graphics, pin images, and thumbnails).
- Post to social media platforms or schedule posts.
- Record or edit video or audio for Reels, TikToks, or YouTube Shorts.
- Write the original long-form content — this skill REPURPOSES existing content only.
- Manage a content calendar beyond the one-week publishing schedule for a single piece.
- Handle paid promotion, boosting, or ad spend allocation.
