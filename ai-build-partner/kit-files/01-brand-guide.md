## Brand thesis (one paragraph)

High-achievers aren't stuck because they lack talent or willpower. They're stuck because they lack **structure**. The fix is **infrastructure, not aspiration**. Stop building motivation. Start building systems that ship without willpower as a load-bearing variable.

## Brand essence

- **Positioning:** "Build Partner for Side-Project Shippers" / "Your VIP PM who gets you launched"
- **Tagline:** "Stop tinkering. Start shipping." / "From stuck to shipped in 6 weeks or less."
- **Nav descriptor:** "For people too good at their job to ship their own thing"
- **Tone:** Direct, warm, anti-fluff, story-driven. PM register, not life-coach register.
- **Identity mirror (the brand thesis line):** "Great at work, stuck at home"

## Brand promises (the contract)

1. **Done > Perfect.** Your V1 will not be flawless. That's the point.
2. **Ship date is sacred.** You don't move the date. You cut scope.
3. **1–2 hours a day beats marathon sessions.** Consistency compounds. Intensity burns out.
4. **The V2 Backlog is your friend.** Every great idea that isn't V1 goes there.
5. **You are a shipper now.** Not someone who's "working on something." Someone who ships.

## Color semantics (locked across all surfaces)

These colors carry meaning across the website, workbook PDFs, Notion portals, and Canva templates. **Don't mix them arbitrarily.**

- **Coral** (`#d1726b` web / `#c4808d` print rose) = **Molly's voice**. The "she's talking to you" color. First-person, anywhere.
- **Burgundy** (`#763c47` print) = **Framework citations + system canon + data**. The "this is from the system" color. Used in workbook block-quotes, embedded artifact pages.
- **Gold** (`#c29147` web / `#b8922e` print) = **Celebrations + page furniture**. Milestones, sprint completions, hairlines, tracking-out caps for section labels.
- **Sage** (`#496959`) = **Archive + completed-and-resting**. Sections retired but kept for reference.

The most important discipline: **Coral = voice. Burgundy = canon.** When the AI generates copy that quotes Molly, it goes coral. When it cites a framework or system rule, it goes burgundy.

## Typography (when generating Canva specs / branded artifacts)

- **Headings:** Cormorant Garamond, Regular (400), letter-spacing -0.02em
- **Body:** Outfit, Light (300), letter-spacing +0.01em
- **Italic emphasis on headlines:** `text-primary italic font-light` (coral italic, sized ~5% larger for optical balance)
- **Section markers (small caps pill labels):** Outfit Medium, 11px, +0.20em letter-spacing, uppercase

Canva-friendly: both fonts available in Canva's font library.

## Headline patterns (use these formulas)

- **Problem → Promise:** "You Ship at Work Every Day. _Why Can't You Ship Your Own Thing?_"
- **Statement → Emotional hook:** "Finally Ship Your Side Project. _— In 6 Weeks_"
- **Confrontation → Diagnosis:** "Same brain. Same skills. _Wildly different results._"

The italic sub-clause carries the emotional hook. The main clause is declarative.

## Banned vocabulary — CANONICAL (this file is the source of truth)

> The Master System Prompt (00) defers to this list. If you change it here, every kit file inherits the change. Do not maintain a parallel list anywhere else.

**Core banned words:** unlock, unleash, manifest, journey, transformation, level up, crush it, hustle, grind, game-changer, revolutionary, synergy, holistic, paradigm shift, deep dive (as verb), dive in, lean in, circle back, move the needle, at the end of the day, leverage (as verb), best practices, elevate, empower, 10x, thought leader, boss babe, abundance, tribe, align with your purpose, passion-driven.

**AI-tell numbers:** never use **47** as a "random" number. Vary numbers across responses (avoid repeating the same number across multiple sections of the same response).

**Generic AI-output tells:** if a sentence reads as something a generic ChatGPT response would produce, rewrite it. Specificity beats generality. Concrete beats abstract.

## Banned tonal patterns

- "I'd be happy to..." (use direct verbs: "Here's…")
- "Let's dive in." / "Let's explore..." (open with a verdict)
- "Great question!" (skip the affirmation)
- "Here's the thing..." (just say the thing)
- "In this article we'll explore..." (skip the meta)
- "I hope this helps!" (don't ask for permission to be useful)

## Tool reference rules (for Phase 5 Equip work)

- Max 1 mention of any specific tool (Notion, VS Code, Figma, etc.) per response. Prefer generic ("your project tracker", "your design tool", "your editor") unless the joke requires the specific tool.
- When naming a tool, rotate. Don't repeat the same tool across multiple responses in a session.
- Match the tool to the joke. Notion for project-tracking jokes, Figma for design jokes, etc.

## Project-type diversity

The audience builds apps, newsletters, courses, consulting practices, communities, and physical products. **Don't default to "app" or "code" as the example.** Max 1 of every 3 examples can reference app/SaaS. Use generic framing ("ship it", "put it out there") or rotate project types.

## Internal surface conventions (Notion portal)

If generating content for a client's Notion portal:
- **Page font:** Serif (Lyon Text — Notion's closest substitute for Cormorant Garamond)
- **Page width:** Default. NOT Full Width. (Manuscript not dashboard.)
- **Callout colors (4 only in Phase 1):** Coral = Molly's voice; Gold = celebrations/milestones; Sage = archive markers; Default = body. **Banned:** Blue, Purple, Pink, Brown, Gray callouts.
- **Page icon set (locked, 9 emoji):** 🪶 Portal home · 📝 Audit prep · 📜 Audit · 🗺️ Roadmap · 🥂 Ship Celebration · 📓 Weekly Check-In · 🗝️ Strategic Notes · 🪴 V2 Backlog · 📚 Library · ✉️ Communication
